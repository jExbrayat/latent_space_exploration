The autoencoder was trained using a variational layer and a customized loss function with a Kullback-Leibler penalization term. This enforces the area of each class to be homogeneous with smooth borders.

Despite the absurdly low dimensionality of the latent space - 2 dimensions against 28 x 28 = 784 for the input data - the decoder manages very well to generate realistic digits, which demonstrates the ability of autoencoders to achieve dimensionality reduction.
